## Scripts

### `opening_price_measures.py` -- Opening Price Sensitivity

**Paper**: Appendix A, Table A1

Tests sensitivity of main results to different definitions of the opening price: first trade, volume-weighted first hour, and 5th percentile lifecycle price.

**Outputs**: `opening_price_robustness.tex`, `opening_price_robustness.json`
